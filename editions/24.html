3 ferramentas para criar e gerenciar projetos web
Cada vez mais um número maior de ferramentas surge para auxiliar e agilizar o desenvolvimento web. Conheça alguns frameworks que têm como objetivo controlar e gerenciar o uso de bibliotecas, pré-processadores e utilitários em sites e aplicações web.

Node.js Inspector
Ferramenta que permite inspecionar seu código em real time, editar seu código Node.js pelo próprio inspector isso mesmo, editar pelo inspector. Vale a pena testar a ferramenta. 

Hint.css
Hint.css é uma biblioteca escrita em SASS, utilizada para criar simples tooltips usando apenas HTML/CSS.

Flight
Flight foi criado pelo Twitter e consiste em um framework lightweight baseado em componentes que mapeia comportamentos a nós na DOM, que usa ES5-shim e jQuery.

TransitionEnd
TransitionEnd é uma lib criada por Evandro Gonçalves (@evandrolg) em JavaScript, agnostica e cross-browser para trabalhar com o evento transitionend.

Voxeljs
Um toolkit open source para voxel games para web browsers modernos.
