Bibliotecas & Ferramentas
Remote Control
Remote Control fornece uma maneira fácil de controlar os slides de apresentações HTML5. Foi desenvolvido utilizando nodeJS e websockets, criado pelo Jaydson (@jaydson) e pelo Felipe Moura (@felipenmoura), agora esse projeto passa a fazer parte da BrazilJS.

A simple, lightweight url parser for JavaScript
Um simples, analisador url lightweight para JavaScript (~ 1,6 Kb minified, ~ 0.6Kb compactado).  Dica enviada pelo Luiz Tanure (@tanure), envie a sua também.

OverAPI
Todos os Cheat Sheets em um só lugar, conheçam o overapi.com. Dica do Gustavo Costa (@GustavoCostaW).

Watch.JS
Watch.JS é uma pequena biblioteca que traz uma série de possibilidades. O projeto permite executar alguma função, quando algum atributo de um objeto observado é modificado. Dê uma olhada nos exemplos para ver como é simples adicionar Watch.JS ao seu código.
