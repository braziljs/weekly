Libs e Ferramentas

Retro.js
Uma lib que usa a Web Audio API para tocar teclado usando uma representação em JSON de partituras de teclado, no site tem exemplo de 3 músicas e o source esta no github

jsPDF
Uma lib para gerar PDF usando JavaScript, com suporte para todos os browsers modernos e um fallback em flash para browsers legados.

Moment.js
Biblioteca JS para analisar, validar, manipular e formatar datas, com suporte a internacionalização.

Node Web Modules
Uma lista de web frameworks para NodeJS organizados pela quantidade de watchers no GitHub

CreateCSS3
Uma interface que auxilia a geração de algumas snippets de CSS3 complexos, o interessante dessa ferramenta é que ela é bem clean, e oferece um help sobre cada propriedade do CSS3 que ele gera, o que pode ajudar a aprender algo do CSS3

Favico.js
Uma lib para lidar com favicons, o interessante é que a lib consegue colocar até um vídeo como favicon e funciona nos browsers mais modernos, menos o IE10 e inferiores.

Template Enginer Chooser
Uma ferramenta que tem uma lista de template engine JavaScript, onde você escolhe o que você precisa ter na engine de template e a ferramenta mostra quais engines tem o que você precisa.

SassME
Essa ferramenta proporciona uma interface para ajudar a descobrir novas cores usando as funções de cores do Sass
