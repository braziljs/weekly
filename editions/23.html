Ferramentas

Stitches
Ferramenta desenvolvida em HTML5 para gerar sprites CSS, e de quebra ainda gera o CSS para o sprite, é bem simples e intuitivo de utilizar.

MathBox
MathBox é uma library desenvolvida em cima do Three.js e tQuery, com a finalidade de renderizar diagramas de alta qualidade no browser, usando WebGL.

Template Chooser
Ainda tem dúvidas sobre qual template JS usar em seu projeto? Template Chooser irá te ajuda nessa hora, com ele você pode filtrar pelas suas necessidades.

Setup.JS
Evandro Leopoldino (@evandrolg) desenvolveu uma library que facilita a criação de um novo projeto em JavaScript por linha de comando. A library já inclui as algumas ferramentas que você pode precisar em seu projeto, como testes, jshint e ferramenta de minificação.

Temas Sublime Text
Está buscando algum tema para seu Sublime Text? O TmTheme tem dezenas de temas para instalação e um editor visual muito bacana que te possibilita a criar novos temas. 

Angular UI Bootstrap
Uma série de componentes baseada no design do Twitter Bootstrap para aplicações usando Angular JS. Bem interessante.
