Breakpoint media-queries
Essa extensão para Google Chrome gera snippets de media-queries baseado no tamanho atual da janela, auxiliando na correção de inconsistências de layout ao usar a técnica de Responsive Design.

Node GH - Todo poder do GitHub no seu terminal
