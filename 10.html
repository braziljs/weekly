Game com GetUserMedia e WebAudioAPI
E mais um experimento super interessante usando getUserMedia saiu da Conferência da W3C! Criado pelo Soyuz, o game usa a detecção de movimento proposta para controlar uma Asa Delta. Também foi  usada a WebAudio API para fazer os efeitos de som estéreo, SVG para o mapa/asa e a biblioteca Crafty.js. O objetivo é que as pessoas possam se guiar pelo som para chegar no objetivo: dar check in na India, França e Brasil, locais dos congressos mundiais da Web em 2011, 2012 e 2013. 

Gerador aleatório de labirintos
Exemplo que utiliza um grafo e um algoritmo de pesquisa depth-first para gerar um labirinto aleatório. Depois disso, é possível encontrar o menor caminho que liga a entrada à saída do labirinto utilizando o algoritmo A Star. Tudo isso utilizando Javascript, jQuery, Underscore, Jasmine para tests unitários e canvas para renderizar o labirinto. Confira aqui o resultado desse exemplo criado por Felipe Lima.
